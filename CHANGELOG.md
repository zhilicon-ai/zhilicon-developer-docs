## [1.1.0] — 2025-06-15

### Added

- Getting started guide with simulator quickstart
- SDK API reference structure (device, model, tensor, profiling)
- Compiler guide: ONNX ingestion, quantization, graph optimization
- Runtime guide: memory management, multi-device, async execution
- Performance guide: profiling, throughput, latency, power optimization
- Hardware overview: ZHI-1 B0 memory map, PCIe interface, eval board setup

### Changed

- Migrated from static HTML to MkDocs Material for versioned, searchable docs

### Added

- Initial public documentation release covering SDK 1.0.x
- Installation, hello inference, and simulator setup guides
