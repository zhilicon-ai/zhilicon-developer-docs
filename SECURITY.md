# Security Policy

## Reporting Security Issues in Documentation

If you find documentation that:
- Incorrectly describes security features or access controls
- Provides guidance that would lead to insecure configurations
- Accidentally includes sensitive or confidential information

Report to: **security@zhilicon.ai** — do not open a public issue.

For SDK security vulnerabilities, see the [SDK Examples SECURITY.md](https://github.com/zhilicon-ai/zhilicon-sdk-examples/blob/main/SECURITY.md).
